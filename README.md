# PYTHON CALCULATOR
A simple yet powerful command-line calculator built using python.This project demonstrates core programming conceppts such as functions,loops and conditionals.

## FEATURES
Addition
Subtraction
Multiplication
Division(with zero division handling)
Power and Modulus
Input validation
Continuous execution until exit

## Technologies Used
Python

## How to Run
Make sure python is installed
Open terminal/command prompt
Navigate to the project folder
Run the command:

```bash
python calculator.py
